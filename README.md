# **Workout Generator**

## **Description**

I created a Workout Generator app that creates a personalized workout plan based on the user's information and preferences. The user selects their gender, enters their age, chooses their fitness goals, fitness level, preferred equipment, and the body parts they want to train. Based on these selections, the program generates a workout routine that is tailored specifically to the user.

## **Features**

- Users can select their gender and enter their age in an input box.
- Users can choose their fitness goals such as strength, endurance, or general fitness.
- Users can select their current fitness level (beginner, intermediate, or advanced).
- Users can choose the equipment they prefer to use.
- Users can select which body parts they want to focus on during their workout.
- The program generates a personalized workout plan based on all the selections.
- Rest time between exercises varies depending on the chosen workout and difficulty.

## **How It Works**

- I used Tkinter to build the user interface, including labels, input boxes, radio buttons, and checkboxes.
- Radio buttons were used for selections where the user should only choose one option, such as gender or fitness level.
- Checkboxes were used for selections where the user could choose multiple options, such as preferred equipment or body parts to train.
- The user's selections are stored in variables, and when the "Generate Workout Plan" button is clicked, the program reads all the inputs.
- Based on these inputs, the program selects appropriate exercises and displays a workout routine tailored to the user.
- The program also adjusts the rest time between exercises depending on the workout intensity and selections made.
